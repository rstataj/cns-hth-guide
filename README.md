# cns-hth-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖各类操作指南、参考信息或工具页面。所有页面均为静态 HTML 文件，不依赖后端服务，可直接在浏览器中打开或通过 GitHub Pages 等静态托管服务访问。

## 项目简介

该仓库的主要目的是集中存放和整理一系列独立 HTML 页面。这些页面可能包含教程、速查表、工具界面或其他形式的信息展示，每个页面均为自包含的 HTML 文件，便于分享和离线使用。

## 目录说明

仓库根目录下直接放置各个 HTML 页面文件。为保持结构清晰，可包含以下类型的文件或目录：

- `*.html` — 独立的 HTML 页面文件
- `assets/` — 存放页面所需的样式表（CSS）、脚本（JavaScript）、图片等静态资源
- `docs/` — 可选的文档或说明文件

每个 HTML 页面应有明确的文件名，以反映其内容主题。建议在页面内部添加简短的标题或描述，便于识别。

## 页面归档说明

- 所有页面均为静态 HTML，不包含动态内容或外部依赖（除非在页面内显式引用 CDN 资源）
- 页面之间保持独立，无相互链接依赖（除非有意设计为系列页面）
- 归档内容不针对任何特定域名或网站，仅为通用信息或工具
- 页面编码统一使用 UTF-8，以确保中文及其他字符正常显示

## 维护说明

- 欢迎提交 Pull Request 添加新页面或改进现有页面
- 新增页面请确保文件命名合理，内容不涉及具体商业推广或敏感信息
- 若页面引用外部资源（如字体、图标库），建议注明来源或使用许可
- 如需删除或修改页面，请通过 Issue 或 PR 说明理由

## 使用方式

1. 克隆本仓库到本地
2. 直接双击打开任意 `.html` 文件，即可在浏览器中查看
3. 或启用 GitHub Pages（Settings > Pages），选择 `main` 分支，即可通过 `https://<用户名>.github.io/cns-hth-guide/` 访问根目录下的页面
